# GPU-Ed25519-Curve25519-pubkey-vanity-prefix-and-suffix
GPU-accelerated Ed25519 vanity address generator with prefix and suffix matching.

-help
Fingerprint search (SHA256 base64):
  --fingerprint-prefix <b64>   e.g. dead
  --fingerprint-suffix <b64>   e.g. cafe

Vanity pubkey search (hex of raw 32-byte Ed25519 pubkey):
  --pubkey-prefix <hex>        e.g. cafe
  --pubkey-suffix <hex>        e.g. baby

  --blocks <n>                CUDA blocks per GPU (default: 256, max: 4096)
  --gpu <id[,id,...]>          GPU IDs to use (default: 0)
                               Example: --gpu 0,1,2,3

Note: --fingerprint-* and --pubkey-* modes cannot be combined.

#Example 
build>ed25519brute_cuda.exe --pubkey-prefix dead --pubkey-suffix beaf --blocks 256 --gpu 0,1,2
